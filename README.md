# Online-Food-Ordering-System--Django-
An end-to-end Food Ordering System developed using Django. Includes user authentication, dynamic menu display, add-to-cart functionality, checkout process, order history, and a custom admin panel for managing food items and order statuses.


# 🍔 Online Food Ordering System (Django)

A complete **Food Ordering Web Application** built using **Django, HTML, CSS, Bootstrap, and SQLite**.  
This project allows users to browse food items, add to cart, place orders, and track order status.

---

## 🚀 Features

### 👤 User Side
- User Registration & Login
- View Food Menu
- Add to Cart
- Checkout System
- Order History
- Order Status Tracking (Pending → Preparing → Delivered)

### 🛠 Admin Panel
- Add Food Items
- Update Food Items
- Delete Food Items
- View All Orders
- Update Order Status

---

## 🧑‍💻 Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite
- **Authentication:** Django Auth System
